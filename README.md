## 판단문
<pre>
판단 [변수]
->판단1 실행1
->판단2 실행2
..
=> #해당없으면 실행

</pre>

<pre>
점수 = 입력('점수 : ')

->0~39 출력해 'F등급'
->40~59 출력해 'D등급'
->60~79 출력해 'C등급'
->80~89 출력해 'B등급'
->90~100 출력해 'A등급'
=> 출력해 '점수 입력 오류!'
끝
</pre>
