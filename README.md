<pre>

나=20

>30
</pre>

<pre>
가=10
나=20
출력하기 가+나*가

>210
</pre>

<pre>
가=정수입력('가:')
나=정수입력('나:')
출력하기 '가+나=',가+나

>가:10
>나:30
>가+나=40
</pre>

<pre>
가,나=20,30
출력하기 '[가]+[나]=[가+나]'

>20+30=50
</pre>

<pre>
